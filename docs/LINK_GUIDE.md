# 교과 간 연결 그래프 — 사용 가이드

커리큘럼 위버의 교과 간 연결 그래프는 서로 다른 교과의 성취기준이 어떻게 연결되는지 시각적으로 탐색하는 도구입니다. **3계층 링크 품질 시스템**으로 검증된 연결과 AI 제안을 명확히 구분합니다.

---

## 1. 3계층 링크 상태

모든 교과 간 연결은 아래 3단계 중 하나의 상태를 가집니다.

| 상태 | 의미 | 기본 표시 |
|------|------|----------|
| **Published** (게시) | 교육적으로 검증된 연결 | 항상 표시 |
| **Reviewed** (검토됨) | 검토는 완료되었으나 아직 미게시 | 토글 ON 시 표시 |
| **Candidate** (후보) | AI가 제안한 검증 전 후보 | 토글 ON 시 표시 |

새로 생성되는 AI 연결은 모두 **Candidate** 상태에서 시작하며, 관리자 검토를 거쳐 Published로 승격됩니다.

---

## 2. 시각적 구분

### 검증된 연결 (Published)
- **실선**으로 표시
- 연결 유형별 **색상** 구분:
  - 교과연계 (주황)
  - 동일개념 (파랑)
  - 적용/활용 (초록)
  - 선수학습 (빨강, 점선)
  - 확장/심화 (보라)
- **100% 불투명도**

### AI 제안 후보 (Candidate / Reviewed)
- **회색 점선**으로 표시
- **40% 투명도** — 배경에 흐릿하게 보임
- 검증된 연결과 한눈에 구분 가능

---

## 3. 그래프 사용법

### 3-1. 교과 선택
1. 데이터 관리 페이지에서 교과 태그를 클릭합니다.
2. **2개 이상** 교과를 선택하면 인라인 그래프가 나타납니다.
3. 선택한 교과 간의 **교차 연결만** 표시됩니다 (같은 교과 내 연결은 제외).

### 3-2. 노드 탐색
- **노드 클릭**: 해당 성취기준의 상세 정보와 연결 목록이 오른쪽 패널에 표시됩니다.
- **노드 호버**: 연결된 노드가 하이라이트됩니다.
- **배경 클릭**: 선택이 해제되고 전체 연결 목록으로 돌아갑니다.
- **초기화 버튼** (우측 상단): 줌을 리셋하고 전체 보기로 돌아갑니다.

### 3-3. AI 제안 보기
- 그래프 상단의 **"AI 제안 포함"** 체크박스를 켭니다.
- 회색 점선으로 표시되는 후보 연결이 추가로 나타납니다.
- 유용한 융합 아이디어를 발견하는 데 활용하세요.

### 3-4. 연결 상세 정보
노드를 클릭하면 각 연결에 대해 다음 정보가 표시됩니다:

| 아이콘 | 필드 | 설명 |
|--------|------|------|
| 💡 | **연결 근거** | 두 성취기준이 연결된 교육적 이유 |
| 🔗 | **융합 주제** | 융합 수업의 중심 주제 (예: "에너지와 환경") |
| 📝 | **수업 아이디어** | 바로 활용할 수 있는 수업 설계 힌트 |

---

## 4. 연결 유형 설명

| 유형 | 색상 | 설명 | 예시 |
|------|------|------|------|
| **교과연계** | 주황 | 서로 다른 교과에서 관련 내용을 다룸 | 과학의 에너지 ↔ 기술·가정의 에너지 효율 |
| **동일개념** | 파랑 | 같은 개념을 다른 교과에서 다룸 | 수학의 함수 ↔ 과학의 그래프 해석 |
| **적용/활용** | 초록 | 한 교과의 내용이 다른 교과에서 적용됨 | 수학의 통계 → 사회의 데이터 해석 |
| **선수학습** | 빨강 | 먼저 학습해야 하는 관계 | 수학의 비례식 → 과학의 농도 계산 |
| **확장/심화** | 보라 | 기본 학습을 심화·확장하는 관계 | 일반 물리학 → 고급 물리학 |

## 5. 관리자 기능

### 링크 상태 변경
관리자는 API를 통해 링크 상태를 변경할 수 있습니다:

```
PATCH /api/standards/links/{linkId}/status
Body: { "status": "published" }
```

유효한 상태값: `candidate`, `reviewed`, `published`

### 데이터 마이그레이션
기존 AI 생성 링크를 Supabase DB로 이관하려면:

```bash
SUPABASE_URL=... SUPABASE_SERVICE_ROLE_KEY=... node scripts/migrateLinksToDB.js
```

## 6. 자주 묻는 질문

**Q: 기본으로 표시되는 연결은 무엇인가요?**
A: `published` 상태의 연결만 기본 표시됩니다. 이 연결들은 교육적으로 검증된 것입니다.

**Q: AI 제안 후보는 어떻게 만들어지나요?**
A: Claude Opus 모델이 성취기준 간 교육적 연관성을 분석하여 생성합니다. TF-IDF 키워드 매칭과 달리 의미적 관련성을 기반으로 합니다.

**Q: 잘못된 연결을 발견하면 어떻게 하나요?**
A: 관리자에게 알려주세요. 관리자가 해당 링크의 상태를 `candidate`로 되돌리거나 삭제할 수 있습니다.

**Q: "AI 제안 포함"을 켜도 새 연결이 안 보여요.**
A: 현재 선택한 교과 조합에 후보 연결이 없을 수 있습니다. 다른 교과 조합을 시도해보세요.

**Q: 수업 아이디어나 융합 주제가 비어있는 연결이 있어요.**
A: 이 필드는 점진적으로 채워지고 있습니다. 향후 AI 업데이트를 통해 모든 연결에 수업 아이디어가 추가될 예정입니다.
